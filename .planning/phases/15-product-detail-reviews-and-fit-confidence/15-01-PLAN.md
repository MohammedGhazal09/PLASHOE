# Plan 15-01 - Product Detail Data, Route, and Related Products

<objective>
Prepare product data contracts and related-product API support for the Phase 15 product detail page.
</objective>

## Tasks

1. Extend `Product.js` with optional `gallery`, `materials`, `careInstructions`, `fitGuide`, `reviewCount`, `ratingDistribution`, and `fitSummary` fields.
2. Extend product validators with bounded optional fields while keeping create/update backward-compatible.
3. Seed realistic materials/care/fit values for local seeded products without requiring real business copy.
4. Add `GET /api/products/:id/related` before `/:id`, validate `limit`, cap at 8, exclude the current product, and implement deterministic same gender/category fallback.
5. Add product-detail backend tests for richer fields and related-products behavior.
6. Extend `productsApi` and `normalizeProduct` for richer fields and related products.
7. Make ProductCard name/image navigate to `/products/:id` while preserving wishlist, quick view, and cart actions.
