# Plan 24-01: Admin Back-in-Stock APIs

## Goal

Add protected admin list, summary, and status update APIs for back-in-stock demand.

## Tasks

- Extend back-in-stock validators for admin query params, params, and status updates.
- Add summary aggregation with status counts, pending size counts, and top product/size demand.
- Add paginated admin list filters by product id, size, email, status, and search.
- Add status transition handler for `notified` and `cancelled`.
- Mount routes behind `protect, admin`.

## Done

Implemented in `Backend/validators/backInStock.js`, `Backend/controllers/backInStockController.js`, and `Backend/routes/backInStockRoutes.js`.
