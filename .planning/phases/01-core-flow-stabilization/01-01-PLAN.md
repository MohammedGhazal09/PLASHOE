# Phase 01 Plan 01: Core Source Contract Fixes

<objective>
Fix the five source-level Phase 1 contract failures in contact submission, authenticated checkout, coupon application, and missing-cart coupon removal. This plan is the implementation wave for CORE-01 through CORE-05 and should leave the existing route paths, response envelopes, auth boundary, visual styling, and local guest cart behavior intact.
</objective>
